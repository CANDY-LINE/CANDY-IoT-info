[CANDY IoT](http://www.candy-line.io/proandsv.html#candyiot)は以下のGPIOピンを使用しています。

| Edison GPIO  | AMP520/AMM570 | CANDY IoT |
| ------------ | ------------- | --------- |
|   49 (OUT)   |     POWER     |     -     |
|   46 (OUT)   |     RESET     |     -     |
|   14 (OUT)   |    RESERVED   |     -     |
|   15 (OUT)   |       -       |    LED1   |
|   40 (OUT)   |       -       |    LED2   |
